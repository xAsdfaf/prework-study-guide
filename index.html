Open the control console to see what is happening!

HTML

  • The head element contains information aobut the webpage.
  • The body element represents the visible content shown to the user.
  • Metadata is defined in the head element, and contains back-end information.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates dow much space we want around the content inside an element.
  • A boxshadow adds a shadow effect around an element's frame. Multiple effects can be set by separating them with commas.

Git

  • git status: checks what branch we are currently working on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin branch-name: pulls all code from the identified branch to the branch currently being worked on

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • "for" loops are a single code block that will repeatedly perform a task on all items in an array.
  • A function is a set of instructions that tell a computer how to perform a task.
  • A function name must always be followed by parentheses.
